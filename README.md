## [Nemaxe](https://github.com/JackEd0/nemaxe)

Nemaxe is web interface that allow users to create and test school exercises.

Curent version: 1.2.5


### Features

 - Login / Register
 - Browser exercises
 - Create/Read/Update/Delete subjects
 - CRUDE classes
 - Manage users
